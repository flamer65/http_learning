## 🛠 Tools You'll Use

- **curl** — Command line HTTP client (pre-installed on Mac)
- **Node.js** — For writing HTTP servers and clients
- **Browser DevTools** — Network tab to inspect real HTTP traffic

- `exercises/` — Hands-on exercises for you to code
