# Lesson 20: Cookies, Sessions & Authentication

In this lesson, you will implement:
1. Cookie setting and reading
2. Session-based authentication
3. JWT-based authentication

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```
2. Run the tests:
   ```bash
   bun test
   ```

Your goal is to make all tests pass by completing the TODOs in the `src` directory.
