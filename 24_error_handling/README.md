# Lesson 24: Error Handling

## Quick Summary
Learn how to gracefully handle asynchronous errors and create a centralized error-handling middleware for an Express app using custom error classes.

## Instructions
1. `cd exercises`
2. `bun install`
3. Implement `src/utils/AppError.ts`, `src/utils/errors.ts`, `src/utils/asyncHandler.ts`, and `src/middleware/errorHandler.ts`
4. Run `bun test` to see your tests pass!

You can check `solutions/` if you get stuck.
