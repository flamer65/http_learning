# Lesson 25 — Capstone: Full REST API

Welcome to the capstone project! You will build a complete "Bookshelf API" that integrates everything you've learned: routing, controllers, services, repositories, authentication, CORS, security, caching, error handling, and testing.

## Overview
You are building a REST API for a Bookshelf application. It manages Users, Books, Authors, and Reviews.

## How to Run the Tests

1. Install dependencies:
```bash
bun install
```

2. Run the tests in watch mode:
```bash
bun test --watch
```

Make all the tests pass! Start with unit tests for services, then move to integration tests. Good luck!
