# Lesson 23: The Database Layer

## Quick Summary
Learn how to persist data using SQLite and `better-sqlite3`, while proving the power of the Repository Pattern by swapping out the in-memory database without changing the service layer.

## Instructions
1. `cd exercises`
2. `bun install`
3. Implement `src/config/database.ts` and `src/repositories/sqlite.repository.ts`
4. Run `bun test` to see your tests pass!

You can check `solutions/` if you get stuck.
