# Lesson 19: Middleware Pipeline

In this lesson, you will learn how to build and structure Express middleware.

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```
2. Run the tests:
   ```bash
   bun test
   ```

Your goal is to make all the tests pass by implementing the middleware and routes in the `src` directory.

> **Tip**: Check `lesson.md` for a refresher on middleware concepts!
