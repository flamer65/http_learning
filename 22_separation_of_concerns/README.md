# Lesson 22: Separation of Concerns

## 🎯 Objectives
- Understand and implement the Route → Controller → Service → Repository pattern.
- Keep HTTP concerns in the Controller.
- Keep business logic in the Service.
- Write tests for each layer independently.

## 🏃‍♂️ How to Run
1. Install dependencies: `bun install`
2. Run the tests: `bun test`
3. Notice there are 3 test files testing different layers!
